# ⚡ Poké Search

A sleek, responsive web application that allows users to search for any Pokémon and instantly view their official artwork, typing, weight, and base stats. 

## 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, CSS Grid/Flexbox for side-by-side layout, and pixelated image rendering for retro artwork.
* **JavaScript (ES6+):** Utilizes modern `async/await` syntax for clean asynchronous data fetching.
* **[PokéAPI](https://pokeapi.co/):** The RESTful API used to gather real-time Pokémon data.

## 🚀 Features
* **Live Search:** Fetch data instantly using a Pokémon's name.
* **Error Handling:** Gracefully handles typos and non-existent Pokémon with `try/catch` blocks and user alerts.
* **Dynamic DOM Manipulation:** Injects stats, types, and images directly into the HTML without reloading the page.
* **Keyboard Support:** Trigger searches simply by pressing the "Enter" key.

## 🧠 What I Learned
Through building this project, I solidified my understanding of JavaScript Promises and the Fetch API. Transitioning from `.then()` chains to `async/await` made the asynchronous logic much cleaner and easier to read. I also practiced combining JSON data retrieval with dynamic UI updates.
